Getting Started

EasyLang runs entirely in your browser. Open the Editor, type your first program, and press Ctrl+Enter or click ▶ Run.

Your First Program

say "Hello, World!"

That's it. No imports, no main function, no semicolons. One line → one output.

Comments

# This is a comment — it won't run say "This will run" # inline comment too

Variables

Create a variable with set ... to .... Variables don't need a type — EasyLang figures it out.

set name to "Alice" set age to 25 set score to 98.5 set active to true set nothing to null
# Reassign anytime set age to age + 1 say age

Variable names are case-sensitive: score and Score are different.

Output

Use say (or print) to print anything to the output console.

say "Hello!" say 42 say 3.14 say true say "Name: " + name # concatenation say "Sum: " + (a + b) # use parens for clarity

Conditionals

Use if / elseif / else / end for branching.

set grade to 72 if grade >= 90 say "A" elseif grade >= 80 say "B" elseif grade >= 70 say "C" else say "Below C" end

Every if must be closed with end. Nesting is supported.

Loops

repeat — Simple repetition

repeat 5 say "Hello!" end

loop from/to — Range loop

loop i from 1 to 10 say i end # Countdown loop i from 10 to 1 say i end # Custom step loop i from 0 to 20 with 5 say i end

while — Condition loop

set n to 1 while n <= 100 set n to n * 2 end say n

stop — Break out of a loop

loop i from 1 to 100 if i == 5 stop end say i end

Functions

Define reusable blocks with define and call them with call.

# Define a function define greet with name say "Hello, " + name + "!" end # Call it call greet with "Alice" call greet with "Bob" # Return a value define multiply with a, b return a * b end set result to multiply with 6, 7 say result # 42 # No parameters needed define divider say "──────────────" end call divider

Functions support recursion — they can call themselves!

Lists

# Create set fruits to list "apple", "banana", "cherry" set empty to list # Access by index (0-based) say fruits[0] # apple say fruits[2] # cherry say fruits[-1] # last item # Modify add "mango" to fruits remove "banana" from fruits # Properties say length of fruits say first of fruits say last of fruits say sorted of fruits say reverse of fruits say join of fruits # Loop over list loop i from 0 to (length of fruits) - 1 say fruits[i] end
